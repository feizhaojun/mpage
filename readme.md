## 使用方法

```
      Page({
        el:'page',  // 放置页面的页面容器的id <div id="page"></div>
        current:'当前页',  // 全局变量
        total:'总页数',  // 全局变量
        callback:function(page){
            // page 点击翻页和页码返回的页数
            // 这里可以根据返回的页码做一些动作，比如请求接口，请求结束后可以改写当前页面全局变量中的 当前页 和 总页数，然后再次调用本方法（Page()）。
        }
      });
```
